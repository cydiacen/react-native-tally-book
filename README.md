# react-native-tally-book
> 自己拿来学习`ReactNative`的练手demo，需求，设计和功能全靠自己补充和想象。
欢迎对`ReactNtive`有兴趣的朋友一起探讨，研究。（目前只支持ios）

####使用方法：
    
使用git clone该项目
```
git clone https://github.com/cydiacen/react-native-tally-book.git
```
然后在文件目录下 运行 `npm install` 指令

这个指令会根据 `package.json` 里的配置信息,安装相应的文件

然后执行ios运行指令（前提是已经安装Xcode）

```
   react-native run-ios 
```
此时会启动ios模拟器，并且在模拟器上运行该项目。
